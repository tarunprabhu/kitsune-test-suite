# Kitsune Test Suite #

This is a fork of [llvm-test-suite](https://github.com/llvm/llvm-test-suite) to
which end-to-end tests for [Kitsune](https://github.com/lanl/kitsune) have been
added.

See [Kitsune/README.md](Kitsune/README.md) for instructions on building and
running the tests in this test suite. That document also contains information
for developers on adding tests for Kitsune to this test suite.

The original readme for the `llvm-test-suite` is [here](README.txt)
